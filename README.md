# zen-of-changes

一个基于 `eframe/egui` 的《易经》随机卦象演示工具。软件通过随机生成上卦、下卦和主爻，展示对应卦象与文本说明。

## 主要功能

- 随机生成三组数值：下卦（1-8）、上卦（1-8）、主爻（1-6）
- 根据上下卦组合生成本卦，并显示卦名与卦符
- 图形化展示六爻（从上到下显示）
- 高亮标记主爻位置
- 展示卦辞、主爻爻辞、易传、系辞传、象传
- 已加入中文字体兼容处理，优先加载系统常见中文字体，减少乱码问题

## 运行环境

- Rust stable（建议较新版本，如 `1.93+`）
- 支持桌面图形环境（Linux/macOS/Windows）

## 安装与运行

```bash
# 1) 检查编译
cargo check

# 2) 启动程序
cargo run
```

## 使用方法

1. 启动后点击 `生成随机卦`
2. 查看随机结果：下卦、上卦、主爻
3. 查看本卦信息：
   - 下卦/上卦名称与符号
   - 本卦名称与符号
   - 六爻图（主爻会高亮）
4. 阅读文本解释：
   - 卦辞
   - 当前主爻对应爻辞
   - 易传、系辞传、象传

## 说明

- 当前卦表数据为演示版本，部分卦辞/爻辞可能是占位文本。
- 若你的系统缺少可用中文字体，界面仍可能出现方块字；可安装常见中文字体（如 Noto Sans CJK）后重试。

## 项目结构

- `src/main.rs`：主程序与界面逻辑
- `Cargo.toml`：依赖配置（`eframe`、`rand`）
